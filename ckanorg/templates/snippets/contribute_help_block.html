If you work with data or Python, the weekly Dev Meetings are a great place to start — open to everyone, no experience required. Drop in to listen, ask questions, or find out where your skills are most useful. If you’re ready to dig in, issues labelled Good for Contribution are a reliable starting point, and the Rockstar Tickets list higher-impact work for those who want a bigger challenge.

{% translate "Need help with CKAN?" %}

Stack Overflow and the CKAN docs cover most common questions. GitHub Discussions is the place for CKAN support requests, project sharing, and questions about CKAN. Gitter is where developers talk in real time. For broader conversations — funding, events, portal operations, and community strategy — head to the CKAN Community Forum.

Found a bug or a documentation error? Open an issue on the issue tracker. For security vulnerabilities, email security@ckan.org directly — do not post publicly.
